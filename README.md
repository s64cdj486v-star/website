# Physiotherapie Golzheim – statische Website

Diese Version ist eine komplett statische Code-Basis: HTML, CSS und ein kleines JavaScript für das mobile Menü. Es wird kein WordPress, kein Build-Prozess und keine Datenbank benötigt.

## Hosting

Den gesamten Ordner auf den Webspace laden. Die Startdatei ist `index.html`.

Geeignet für IONOS Webspace, Netlify Drop, Cloudflare Pages, GitHub Pages oder jedes normale FTP/SFTP-Hosting.

## Inhalte ändern

Texte stehen direkt in den HTML-Dateien. Bilder liegen in `assets/img/` und können ersetzt werden. Die Dateinamen sollten gleich bleiben, dann muss kein HTML angepasst werden.

## Design

Diese Version ist bewusst seriöser und medizinischer gestaltet: klare Typografie, ruhige Farben, dezente Flächen, weniger Show-Effekte und stärkerer Praxis-/Vertrauenscharakter.

- `behandlungen.html` Leistungsübersicht

## Variante ohne Logo
Diese Variante entfernt die Icon-/Logo-Marke im Header. Die Praxisbezeichnung bleibt als reine typografische Wortmarke erhalten.
